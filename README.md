# Student Record Manager

A console-based Student Record Manager built using Python.

## Features

* Add Student
* View Students
* Search Student
* Update Student
* Delete Student
* Find Topper
* Calculate Average CGPA
* View Branch-wise Student Count
* Sort Students by Name, Roll Number, and CGPA
* Save Student Data to File
* Load Student Data Automatically

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* File Handling
* Git
* GitHub

## Project Structure

student_record_manager.py

functions.py

students.txt

## How to Run

1. Clone the repository
2. Open the project folder
3. Run:

python3 student_record_manager.py
